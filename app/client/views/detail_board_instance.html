Board informations

  • Description

    • Renesas R-Car M2 Porter is a board specifically designed for automotive infotainment applications. The hardware contains interfaces seldom found on hobbyist boards such as IEBus (automotive 2-wire protocol to connect multiple media devices), EXIO connector, and an Ethernet AVB bus. Video input would allow you to use some rear camera for example, LVDS and the touchscreen header a touchscreen display.
